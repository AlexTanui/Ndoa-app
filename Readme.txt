# NDOA      Online Marriage Registration System



## Description
The Online Marriage Registration System is a web-based application designed to streamline and simplify the process of registering marriages online. It provides an efficient and user-friendly platform for couples to submit their marriage registration details and for staff members of the Office of Registrar of Marriages to manage and process these registrations.

## Features
- User Registration: Couples can create accounts on the system to initiate the marriage registration process.
- Data Management: Staff members can manage the registration data, including adding, updating, and deleting information as required.
- Document Generation: The system enables the generation of marriage registration documents and receipts.
- Order Status Tracking: Couples and staff members can track the status of the marriage registration, whether it is pending, approved, or rejected.
- Authentication: The system ensures secure authentication for registered users.
- User-Friendly Interface: The system is designed with a user-friendly interface to make it easy for couples and staff members to navigate and interact with the system.
 
## Installation
1. Clone the repository to your local machine.
2. Set up a web server (e.g., Apache) and configure it to run the application.
3. Import the database schema and data from the provided SQL file.
4. Update the database connection settings in the configuration file.
5. Access the application through the web server using the provided URL.

## Usage
1. Couples: 
   - Create an account on the system.
   - Fill in the required details for marriage registration.
   - Submit the registration form.
   - Track the status of the registration.
2. Staff Members:
   - Log in to the system using the provided credentials.
   - Manage the marriage registration data, including approval and rejection of registrations.
   - Generate marriage registration documents and receipts.
   - Track the status of registrations.
